Esse projeto faz parte de um estudo sobre classe abstrada. É um exemplo simples de Conta Bancária (Account) não poderá ser instanciada. Contudo poderá ser utilizado as classes que extendem Account, Conta Empresa (Business Account) e Conta Poupança (Savings Account). O programa possui instâncias manualmente criadas afim de testar o funcionamento das classes. Para isso é criado um Array List do tipo Account, que irá receber instâncias dos tipos supracitados. Também demonstando a eficiência do polimorfismo.
